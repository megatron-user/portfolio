# Portfolio

A single-page portfolio site for Shreesh Ramohally Umesh, showcasing data analytics and AI automation
projects (marketing analytics, sports analytics, and AI agent design).

Built as static HTML, CSS, and JS, deployed to GitHub Pages via GitHub Actions.

Live site: `https://megatron-user.github.io/portfolio/` (enable GitHub Pages under repo Settings, Pages,
Source: GitHub Actions, to activate).

To preview locally:

```bash
python3 -m http.server
```

Then open `http://localhost:8000`.
